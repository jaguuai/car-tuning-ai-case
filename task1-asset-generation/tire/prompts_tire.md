# Task 1 — Asset Generation

Same prompt skeleton as the Rim category (background, lighting, framing,
no-logo rule) reused here for consistency across all asset categories.
The camera angle is different from Rim on purpose: a straight-on side
profile view is used instead of a 3/4 angle, since the tread pattern —
the tire's most distinctive visual feature — is only clearly visible
from this angle.
